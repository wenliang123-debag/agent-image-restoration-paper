A curated, actively maintained reading list for **agentic / LLM- or VLM-orchestrated image restoration** (denoising, deblurring, low-light enhancement, artifact removal, and super-resolution), with concise summaries and BibTeX citations.

> This repository **does not host** third-party PDFs. It links to official arXiv pages and stores original summaries/notes.

## Contents
- [`PAPERS.md`](./PAPERS.md): curated paper list (with summaries + arXiv links)
- [`references.bib`](./references.bib): BibTeX entries for citation
- `notes/` (optional): one file per paper (deep-dive notes)

## How to cite
If you use this repository as a resource, please cite the original papers. BibTeX entries are in [`references.bib`](./references.bib).

## Contributing
PRs are welcome. Please keep additions consistent:
- Add the paper to `PAPERS.md` (short summary + tags + arXiv link)
- Add the BibTeX entry to `references.bib`
- (Optional) Add a detailed note under `notes/`
